# Capturas de la guía de ayuda

Pegá aquí capturas reales con estos nombres para que aparezcan en `../index.html`:

- `hud.png` — barra superior (HUD).
- `lateral.png` — barra lateral (Controles y Filtros).
- `centro_tecnico.png` — Centro Técnico ATSEP.

Si un archivo no existe, la guía muestra un marcador en su lugar (no rompe nada).
Cualquier nombre adicional referenciado en `index.html` sigue la misma convención.
